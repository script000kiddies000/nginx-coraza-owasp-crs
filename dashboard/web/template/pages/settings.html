Real IP Extraction

Nginx hanya memakai header di bawah jika koneksi langsung ke WAF berasal dari IP yang dicantumkan di trusted_proxies — ini bukan opsi “custom header” saja: tanpa daftar proxy, header diabaikan dan log tetap menampilkan IP hop terakhir.

Contoh: cloudflared di mesin lain → isi IP yang terlihat sebagai remote_addr (mis. 192.168.x.x). Cloudflared loopback ke nginx → 127.0.0.1 dan ::1. Pisahkan dengan koma; CIDR didukung (mis. 10.0.0.0/8).

IP atau subnet sumber koneksi TCP ke nginx (bukan IP pengunjung). Kosongkan saat simpan = pakai default 127.0.0.1 dan ::1.
